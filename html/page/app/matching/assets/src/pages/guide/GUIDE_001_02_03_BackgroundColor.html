Background Color
배경 색상에만 사용하는 컬러입니다. 텍스트, 배경, 아이콘 색상과 중복되는 색상이 있어도 지정된 토큰 색상으로만 사용합니다.

구성요소 예시

Color/bg/Primitive

브랜드 색상, 컴포넌트를 구성하는 색상입니다.
토큰 설명 참조값 Hex 컬러
Primary 브랜드 주 색상 (예시 : 버튼, 입력 폼) brand/600 #FF8120
primary-press 브랜드 주 색상 (Press, Select) brand/800 #EE6002
secondary 컴포넌트 내 기본 블랙 색상 neutral/200 #EEEEEE
secondary-press 컴포넌트 내 기본 블랙 색상(Press, Select) neutral/300 #E0E0E0
selected 선택되었을 때 배경 색상(예시 : 버튼, 입력 폼) brand/50 #FFF5E6
secondary-press 선택되었을 때 배경 색상(Press, Select) brand/200 #FFC77D

Color/bg

전체 화면에서 자유롭게 사용하는 색상입니다.
토큰 설명 참조값 Hex 컬러
Primary 기본 배경 색상 Base/whie #FFFFFF
secondary 두번째 기본 배경 색상 neutral/50 #FAFAFA
